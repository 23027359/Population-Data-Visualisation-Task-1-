
Task 01 – Population Data Visualization

Objective
Create a *horizontal bar chart* to visualize the population sizes of the top 10 most populous countries as of December 18, 2025, using latest estimates from Worldometers and World Population Review (UN data).

Dataset
Approximate populations (millions) for top 10 countries (late 2025):
- India: 1,464
- China: 1,416
- United States: 347
- Indonesia: 286
- Pakistan: 255
- Nigeria: 238
- Brazil: 213
- Bangladesh: 176
- Russia: 144
- Ethiopia: 135

(Source: Worldometers.info & World Population Review)

Tools Used
- Python (3.x)
- Pandas – for data manipulation and DataFrame creation
- Matplotlib – for generating the bar chart

Requirements
pip install pandas matplotlib

Visualization
A horizontal bar chart shows population distribution, sorted by population size. Key features:
- Bars sorted in descending order of population
- X-axis: Population in millions
- Y-axis: Country names
- Title: "Top 10 Most Populous Countries (Dec 2025)"

How to Run
1. Save as `task01_visualization.py`
2. Run: `python task01_visualization.py`

Highlights India and China's dominance in global population.
